## 3. 代码规范

### 3.1 Python

- 遵循 PEP 8
- 函数名：`snake_case`，类名：`PascalCase`
- 所有路由函数必须包含 docstring
- 数据库操作必须使用 SQLAlchemy ORM，禁止拼接 SQL 字符串
- 异常处理：业务异常用 `HTTPException`，系统异常记录日志后返回 500

### 3.2 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| API路由文件 | `{domain}_router.py` | `material_router.py` |
| 路由前缀 | `/api/{domain}` | `/api/materials` |
| 编号前缀 | 实体缩写 | HC(耗材), RK(入库), LY(领用) |
| 数据库表 | 复数小写 | `materials`, `suppliers` |
| 状态枚举 | 小写下划线 | `pending`, `delivered` |

### 3.3 提交规范

```
feat: 新增入库登记功能
fix: 修复库存校验边界条件
docs: 更新API接口文档
refactor: 重构审批状态机
```

## 4. API设计规范

### 4.1 URL设计

```
GET    /api/materials            # 列表
GET    /api/materials/{id}       # 详情
POST   /api/materials            # 创建
PUT    /api/materials/{id}       # 更新
PUT    /api/materials/{id}/adjust-stock  # 子资源操作
```

### 4.2 响应格式

```json
// 分页列表
{"items": [...], "total": 100, "page": 1, "page_size": 20, "total_pages": 5}

// 操作结果
{"message": "入库成功，单号：RK202606150001", "success": true}

// 错误
{"detail": "库存不足"}
```

### 4.3 状态码

| 状态码 | 含义 |
|:--:|------|
| 200 | 成功 |
| 400 | 业务错误（库存不足、状态不允许） |
| 401 | 未认证 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 500 | 服务器错误 |

## 5. 安全规范

1. **密码**：使用 PBKDF2-SHA256 哈希存储，禁止明文
2. **Token**：JWT 有效期 8 小时，过期需重新登录
3. **权限**：每个 API 必须通过 `require_role()` 装饰器校验角色
4. **日志**：所有关键操作必须记录 `audit_logs`
5. **软删除**：供应商、入库记录使用软删除标记，不物理删除
6. **数据不可删除**：领用申请、操作日志禁止删除

## 6. 部署指南

### 6.1 开发环境

```bash
cd backend
pip install -r requirements.txt
python main.py
# 访问 http://localhost:8000
```

### 6.2 生产环境（MySQL）

```bash
# 1. 创建数据库
mysql -u root -p -e "CREATE DATABASE material_mgmt CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 2. 执行建表脚本
mysql -u root -p material_mgmt < database/schema.sql

# 3. 导入种子数据
mysql -u root -p material_mgmt < database/seed.sql

# 4. 修改 config.py 中的 DATABASE_URL
# DATABASE_URL = "mysql+pymysql://user:pass@localhost:3306/material_mgmt"

# 5. 安装MySQL驱动
pip install pymysql

# 6. 启动
python main.py
```

### 6.3 生产部署（推荐）

```bash
# 使用 gunicorn + uvicorn workers
pip install gunicorn
gunicorn main:app -w 4 -k uvicorn.workers.UvicornWorker -b 0.0.0.0:8000
```
